# Ships at Dawn 🚀

A collection of daily prototypes — small, focused proof-of-concepts built fast.

## What's Here

Each folder is a standalone prototype. Some are polished, some are rough sketches. All were built in a day or less.

The goal isn't perfection — it's exploration. Every prototype answers a question, tests an idea, or scratches a curiosity.

## Structure

```
/prototypes
  /001-project-name
  /002-another-one
  ...
```

Each prototype includes:
- Working code (usually frontend, sometimes full-stack)
- Brief README with what it does and why

## Tech Stack

Varies by prototype, but commonly:
- Vue / React / Vanilla JS
- Firebase (hosting, auth, Firestore)
- AI-assisted development (Claude, Gemini)

## Philosophy

**Ship fast, learn faster.** These aren't production apps — they're experiments. The best way to understand something is to build it.
